Имя папки Автора/Название скина
В данном случае написать вместо Full слово Mini в нужной строке.
РЕДАКТИРОВАТЬ ТОЛЬКО NOTEPAD++ http://notepad-plus-plus.org

ПРИМЕР:
 {
    // ВЫБОР КОМПОНЕНТОВ СКИНА (ССЫЛКИ)  |  SKIN COMPONENTS SELECTION (LINKS)
    ${"PMOD/skins/Armagomen/full/battleChat.json"},       // Информер результатов о завершенном бое в чат / Battle Chat - Result Informer
    ${"PMOD/skins/Armagomen/mini/sessionStatistic.json"}, // Сессионная статистика / Session Statistic
    ${"PMOD/skins/Armagomen/full/systemMessages.json"}    // Обработка сообщений в системном канале | System Channel Messages
 }
